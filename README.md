# Telegram AI Bot с оплатой TON

## Описание
Телеграм-бот с AI (OpenAI) и оплатой через TON-кошелек.  
Главное меню с кнопками /start, /buy и /confirm.  

## Установка
1. Клонировать репозиторий
2. Создать `.env` с BOT_TOKEN и OPENAI_API_KEY
3. Установить зависимости:
   ```bash
   pip install -r requirements.txt
